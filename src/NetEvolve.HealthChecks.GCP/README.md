# NetEvolve.HealthChecks.GCP

[![NuGet](https://img.shields.io/nuget/v/NetEvolve.HealthChecks.GCP?logo=nuget)](https://www.nuget.org/packages/NetEvolve.HealthChecks.GCP/)
[![NuGet](https://img.shields.io/nuget/dt/NetEvolve.HealthChecks.GCP?logo=nuget)](https://www.nuget.org/packages/NetEvolve.HealthChecks.GCP/)

This bundle package provides health checks for various GCP (Google Cloud Platform) services. For specific information about the health checks, please refer to the documentation of the individual packages.

:bulb: This package is available for .NET 8.0 and later.

## Supported GCP Services

- [Google Cloud Firestore](https://www.nuget.org/packages/NetEvolve.HealthChecks.GCP.Firestore/)

## Installation
To use this package, you need to add the package to your project. You can do this by using the NuGet package manager or by using the dotnet CLI.
```powershell
dotnet add package NetEvolve.HealthChecks.GCP
```
